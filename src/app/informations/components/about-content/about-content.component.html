Structural studies of RNA molecules often require insight into their secondary and tertiary structures and an appropriate visualization of the structural information. Contemporary science provides numerous tools to extract base-pairing information (2D structure) from the atom coordinates of RNA molecules. They adhere to distinct internal stereochemical rules for hydrogen bond formation, accept diverse input data formats, and produce output with varying content and syntax. Similarly, there is a variation in the back-end methodology, functionality, and processed data formats between programs that visualize the RNA structure.

RNApdbee integrates various popular secondary structure annotation and visualization tools, provides a unified user-friendly interface to run them, and allows comparison of their performance for the same input data. The web server offers three usage scenarios. The first is to process atom coordinates (PDB or PDBx/mmCIF format) and provide a clear visualization and description of all the interactions found. The second scenario allows transformation between various representations of the 2D structure. In the third scenario, users run all annotation tools simultaneously for the same input data to collectively analyze the results. Additionally, RNApdbee provides the REST API for batch processing.
